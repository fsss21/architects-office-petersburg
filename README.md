**01.10 Классический Петербург (универсальный каталог)**

Современное React приложение для изучения работ выдающихся архитекторов.

## Технологии

- ⚛️ React 18
- 🧭 React Router DOM v6
- ⚡ Vite
- 🎨 CSS3

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173)

## Сборка

```bash
npm run build
```

## Роутинг

Маршруты определены в `src/App.jsx`:
- `/` - Главная страница с прогресс-линией и галереей
- `/biography` - Страница биографий архитекторов

## Функциональность

### Главная страница
- Видео заставка при открытии приложения
- Прогресс-линия с 4 кликабельными точками
- Текстовый блок слева
- Галерея фотографий справа с навигацией
- Кнопки "Назад" и "Главное меню"

### Страница биографий
- 3 кнопки с архитекторами
- Кнопка "Архитектурные принципы и сравнения"
- Кнопка "Назад"
- Блок с биографией выбранного архитектора
- Галерея фотографий с полноэкранным просмотром
- Навигация по фотографиям стрелками

## Требуемые файлы

Для корректной работы приложения необходимо добавить в папку `public/`:
- `preview.mp4` - видео заставка
- Фотографии в папке `public/photos/` (см. структуру выше)
